# Flow BK SMART SMK

1. Admin membuat user dan kelas.
2. Guru BK memasukkan data siswa.
3. Guru/wali kelas mencatat presensi dan pelanggaran.
4. Sistem menghitung poin pelanggaran.
5. Guru BK membuat SP berdasarkan total poin.
6. Konseling, pemanggilan orang tua, dan home visit dicatat.
7. Kepala sekolah melihat dashboard dan laporan.
